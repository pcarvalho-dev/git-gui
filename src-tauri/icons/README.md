# Ícones da Aplicação

Para gerar os ícones necessários:

1. Crie um arquivo PNG de 1024x1024 pixels chamado `app-icon.png` na raiz do projeto
2. Execute o comando:
   ```bash
   npx @tauri-apps/cli icon app-icon.png
   ```

Ou baixe um ícone de exemplo e converta para ICO usando uma ferramenta online como:
- https://convertio.co/png-ico/
- https://www.icoconverter.com/

Para desenvolvimento rápido, você pode usar qualquer ícone ICO básico temporariamente.
